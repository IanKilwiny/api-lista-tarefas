
# Lista de Tarefas Simples
`Projeto em andamento`

Lista de Tarefas Simples que possui um usuário `User` e as tarefas `Task`.


## Tecnologias utilizadas
- Java 17
- Spring Framework
- JPA/Hibernat
- Maven
- H2DataBase
- Flyway 


## Funcionalidades

- Cadastra usuário
- Cria tarefa
- Mostra tarefas
